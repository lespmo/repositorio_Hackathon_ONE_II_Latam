# NoCountry – Data Science

Este repositorio contiene el desarrollo del área de **Data Science** del proyecto NoCountry.  
Aquí se construyen los procesos de análisis, limpieza, modelado y generación de insights a partir de los datos del proyecto.

---

##  Objetivo
Desarrollar soluciones de ciencia de datos que aporten valor al producto, incluyendo:
- Análisis exploratorio de datos
- Limpieza y transformación de datos
- Modelos predictivos o analíticos
- Entregables reutilizables por backend

---

## Tecnologías
- Jupyter Notebook / JupyterLab
- SQL / APIs externas
