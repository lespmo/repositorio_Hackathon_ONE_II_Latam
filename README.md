# FlightOnTime — Predicción de Retrasos de Vuelos

Proyecto desarrollado en el marco de un hackatón.

## Descripción
FlightOnTime es una solución predictiva que estima si un vuelo despegará a tiempo o con retraso,
utilizando datos históricos y un modelo de machine learning.
